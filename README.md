# PT. Akses Artha Media (AAM) - Official Website

Connecting Indonesia, One Satellite at a Time. Professional VSAT and satellite communication solutions.

## 🚀 Overview
This repository contains the source code for the PT. Akses Artha Media official website. The site is built with modern web standards, focusing on high-end aesthetics, responsiveness, and performance.

## 📁 Project Structure
The project follows a standard professional directory structure:
- `/index.html`: Entry point of the website.
- `/pages`: Sub-pages (Products, Services, FAQ, etc.).
- `/assets`:
    - `/css`: Stylesheets and design system.
    - `/js`: JavaScript logic and animations.
    - `/img`: High-resolution optimized images and assets.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom design system, Glassmorphism, and Animations.
- **JavaScript (Vanilla)**: Interactivity and scroll effects.
- **FontAwesome**: Icon system.
- **Google Fonts**: Typography (Poppins & Lato).

## 🎨 Design System
- **Primary Blue**: `#1a4d8f`
- **Secondary Orange**: `#ff6b35`
- **Text Dark**: `#333333`

## 👨‍💻 Development
To run this project locally, simply open `index.html` in any modern web browser or use a local live server (e.g., Live Server extension for VS Code).
